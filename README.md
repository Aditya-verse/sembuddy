### 📖 Overview
SemBuddy is a student-focused academic platform designed to simplify semester preparation. We bridge the gap between chaotic study materials and efficient learning through a modern, responsive interface.

</div>

## 🚀 Key Features
* 📂 **Resource Hub** – Organized notes, syllabi, and previous papers.
* 🤖 **AI Tutor** – Integrated Gemini AI to solve complex queries instantly.
* 📱 **Modern UI** – Clean, intuitive, and fully responsive navigation.
* ⚡ **Fast Performance** – Built on Next.js for a seamless user experience.

## 🛠️ Local Development

Follow these steps to get your own instance of SemBuddy running locally.

### 1. Prerequisites
Ensure you have **Node.js** (v18+) installed on your machine.

### 2. Installation
```bash
# Clone the repository
git clone [https://github.com/yourusername/sembuddy.git](https://github.com/yourusername/sembuddy.git)

# Enter the directory
cd sembuddy

# Install dependencies
npm install
